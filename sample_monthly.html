Chapter 01
이달의 5대 AI 트렌드
4월을 관통한 굵직한 변화. 각 트렌드가 제조 AX 현장에 의미하는 바를 정리했습니다.
01
피지컬 AI, 실험에서 대규모 전환으로 →
Deloitte에 따르면 현재 5%만 피지컬 AI 전환 중이나 3년 내 41%가 대규모 전환 예상. 4월 13일 후속 보고서에서는 글로벌 리더 3,200명 중 58%가 이미 피지컬 AI 운영 중이고 2년 내 80%로 확대 전망. 자동차부품 제조 현장 로봇 도입 타이밍 판단 근거.
02
에이전틱 AI 스프롤 — 94%가 우려 →
OutSystems 조사: 96%가 이미 에이전트 활용 중이나 94%가 "부서별 난립 → 중복 투자 데이터 사일로 거버넌스 공백"을 우려. Microsoft McKinsey가 일제히 2026을 '에이전트 운영 원년'으로 선언.
03
NVIDIA Physical AI Data Factory Blueprint 공개 →
합성 데이터 생성 강화학습 평가를 통합한 오픈 블루프린트. FieldAI Hexagon Uber Teradyne Robotics 채택. FANUC ABB YASKAWA KUKA가 Omniverse Isaac을 가상 시운전에 통합. 서연이화 AX 로드맵 재검토 필요 시점.
04
Audi IRIS, Q2 양산 투입 — Tier 1 신호탄 →
Audi의 비전검사 AI가 2분기부터 VW 그룹 10개 공장에서 양산 라인 투입. 현장 엔지니어가 직접 학습 튜닝 가능한 설계. Bosch Magna는 장비 고장 예측 시퀀싱 플랫폼 투자 확대. OEM의 메시지는 "AI 없는 공급사는 2027 수주 경쟁 후보군에서 빠질 수 있다."
05
국내 AI 주류화 — 다날 AAIF, 현대차 대전환 →
다날이 한국 기업 최초로 에이전틱 AI 재단(AAIF)에 합류해 구글 OpenAI와 AI 결제 표준 정립 참여. 현대차는 "AI 로보틱스 기업"으로 대전환 선언. 국내 기업 85%가 생성형 AI 도입, 10곳 중 8곳이 예산 확대 (CIO 조사).

Chapter 03 Guide
서연이화 AX 실행 체크리스트 7선
4월 뉴스레터에서 반복 제기된 실행 과제를 1장으로 압축. Q2 PoC 킥오프 전 점검 필수.

✓  데이터 표준 API 호환성 — 에이전트 간 데이터 형식 통일 점검했는가?
✓  거버넌스 프레임워크 — 도입 승인 보안 KPI 3개 기준 사전 정의했는가?
✓  PoC→확산 기준 — 정량 성공 지표와 단계별 로드맵 작성했는가?
✓  현업 전담 1인 + PoC 4~8주 타임박스 — 배치했는가?
✓  Q2 PoC → Q3 검증 → Q4 표준화 → 2027 확산 — 분기 로드맵 확정했는가?
